- Porta destinada: 8080
- Necessária ter instalado na máquina o [MySql](https://www.mysql.com/downloads/), e efetuar a alteração das credenciais em resources -> application.properties.
***

1. Para rodar a aplicação, digite o seguinte comando:

`$ gradle bootRun`
***

2. Após iniciar a aplicação, acesse a documentação através deste link:

http://localhost:8080/swagger-ui.html
